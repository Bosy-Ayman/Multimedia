# Multimedia Processing (DSAI 413) - Spring 2025-2026

Welcome to the repository for the **Multimedia Processing (DSAI 413)** course.

---

## 📂 Repository Structure

- **`Assignments/`**: Solutions and code for multimedia processing assignments.
- **`Code/`**: Standalone scripts and Jupyter Notebooks explored during the course.
- **`Notes/`**: Detailed markdown notes from lectures.
  - **`assets/`**: Images and diagrams referenced in the notes.
- **`Questions/`**: Examples, quizzes, and practice questions.

## 🚀 Overview
This course covers various topics in multimedia processing including Space Alignment, Diffusion Models, Neural Radiance Fields (NeRF), Colpali, and Vector Databases.
